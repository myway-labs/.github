# MyWay HQ

> Проектирование надежной бэкенд-архитектуры, масштабируемых API и AI-решений для бизнеса.

Создаем отказоустойчивые серверные системы, автоматизируем сложные бизнес-процессы и интегрируем современные технологии искусственного интеллекта в существующую инфраструктуру клиентов.

---

## 🛠 Направления разработки & Услуги

*   **Backend & API Development** — проектирование серверных приложений, разработка RESTful API, оптимизация баз данных.
*   **AI Integration & Automation** — создание автономных AI-агентов, ботов с RAG-архитектурой, интеграция LLM.
*   **Web Services & PWA** — разработка веб-приложений (PWA), интеграция сторонних сервисов/API-платформам.
*   **Architecture & Refactoring** — аудит текущего кода, исправление архитектурных ошибок, миграция данных и повышение безопасности систем.

---

## 💻 Технологический стек

Используем проверенные временем и индустрией технологии для создания стабильного софта:

*   **Core / Languages:** Python (FastAPI, Django, Asyncio)
*   **Databases:** PostgreSQL, Redis, Vector DBs (для ИИ-решений)
*   **DevOps & Infra:** Docker, Docker Compose, CI/CD (GitHub Actions / GitLab CI)
*   **AI / Data:** OpenRouter API, LangChain, RAG-системы

---

## 🤝 Информация для заказчиков и партнеров

*   **Контроль и прозрачность** — ведем разработку в изолированных приватных репозиториях с доступом к коду и документации.
*   **Независимость кода** — после завершения проекта и финального расчета, права владения репозиторием официально передаются на аккаунт заказчика (*Transfer ownership*).
